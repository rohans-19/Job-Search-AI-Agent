# Job-Search-AI-Agent
AI-powered career assistant that automates job searching, optimizes resumes for ATS systems, provides company insights, and conducts mock interviews
In this project, we developed a Python-based job search chatbot. We explored web scraping techniques for job platforms like Indeed and Naukri (though facing challenges) and successfully integrated with the Adzuna API to fetch job listings. The chatbot provides an interactive way to search for job roles and locations, displaying key details for prospective candidates.
